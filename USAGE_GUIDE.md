# ColorLight LED 텍스트 시스템 사용 설명서

이 시스템은 라즈베리파이와 ColorLight 제어 카드를 사용하여 LED 패널에 숫자를 실시간으로 표시하는 장치입니다. 외장 숫자 키패드를 사용하여 누구나 쉽게 조작할 수 있도록 설계되었습니다.

---

## 1. 하드웨어 준비
1.  **라즈베리파이**: 라즈베리파이 OS Lite 버전이 설치된 기기.
2.  **ColorLight 제어 카드**: 5A-75B 또는 호환 카드와 LED 패널 연결.
3.  **네트워크 케이블**: 라즈베리파이 이더넷 포트와 제어 카드 연결.
4.  **외장 키패드**: 라즈베리파이 USB 포트에 숫자 키패드 연결.

## 2. 프로그램 설치 (최초 1회)
라즈베리파이의 터미널에서 `C` 폴더로 이동한 후 다음 스크립트를 실행합니다.

```bash
cd ~/LED-Text/C
chmod +x setup_autostart.sh
./setup_autostart.sh
```
*이 과정이 완료되면 이제 기기를 켜기만 하면 자동으로 프로그램이 실행됩니다.*

## 3. 조작 방법 (실시간 입력)
프로그램이 실행되면 화면에 `[출력할 텍스트 입력]:` 이라는 메시지가 뜹니다.

### 🔢 숫자 입력
*   **기본 입력**: 키패드에서 원하는 숫자를 누르고 `Enter`를 누르면 즉시 화면에 표시됩니다.
*   **3자 제한**: 숫자를 여러 개 입력해도 **마지막에 누른 3개 숫자**만 화면에 큼직하게 표시됩니다. (예: `1234` 입력 시 `234` 표시)
*   **NumLock 무관**: 키패드의 `NumLock` 불이 꺼져 있어도 정상적으로 숫자가 입력됩니다.

### 🧹 화면 지우기 (초기화)
*   **`+` 버튼**: 키패드의 플러스(`+`) 키를 누르고 `Enter`를 누르면 화면의 숫자가 사라지고 깨끗한 검은색 배경이 됩니다.

## 4. 고급 설정 변경 (`LED_Config`)
글자 크기나 색상을 바꾸고 싶을 때는 `C/LED_Config` 파일을 수정하세요.

*   **`font_size`**: 글자 크기 (기본 200)
*   **`text_color`**: 글자 색상 (`white`, `red`, `green`, `blue` 등)
*   **`bg_color`**: 배경 색상 (`black` 권장)
*   **`brightness`**: 밝기 (0 ~ 100 사이 숫자)

## 5. 자주 묻는 질문 (Q&A)

**Q: 화면에 아무것도 안 나와요.**
> A: 라즈베리파이와 제어 카드 사이의 랜케이블 연결을 확인하세요. 또한 `LED_Config`의 `interface` 항목이 `eth0`(또는 실제 랜포트 이름)로 되어 있는지 확인하세요.

**Q: SSH로 접속했는데 프로그램이 안 뜹니다.**
> A: 이 프로그램은 보안 및 편의를 위해 라즈베리파이에 연결된 **모니터 화면(TTY1)**에서만 자동으로 실행되도록 설정되어 있습니다. SSH 접속 중에는 수동으로 `./colorlight_led`를 입력하여 실행할 수 있습니다.

**Q: 프로그램을 종료하고 싶어요.**
> A: 키보드에서 `Ctrl + C`를 누르면 프로그램이 종료되고 일반 터미널 화면으로 돌아갑니다.
